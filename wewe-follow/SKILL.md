# 公众号更新追踪

追踪微信读书书架上订阅的公众号，通过对比本地状态检测是否有新文章更新。

## 目录结构

```
wewe-follow/
├── SKILL.md                  # Skill 说明文档
├── scripts/
│   ├── check.py              # 检查公众号更新状态
│   ├── detail.py             # 查看单个公众号详情
│   ├── .env.example          # 环境变量模板
│   └── data/
│       └── last_check.json   # 上次查询快照（自动维护）
```

## 核心脚本

| 脚本 | 触发方式 | 职责 |
|------|---------|------|
| `check.py` | agent 调用 | 查书架 → 过滤公众号 → 对比本地状态 → 输出更新摘要 |
| `detail.py` | agent 调用 | 查看指定公众号的详细信息（简介、更新时间、阅读进度） |

## 业务流

### 1. 检查更新

**触发词**：`检查公众号` / `公众号更新` / `有没有新文章` / `公众号`

**工作流**：

1. 执行 `python3 scripts/check.py`，获取 JSON 输出
2. 按以下格式展示结果：

```
📚 公众号更新检查（共 2 个）

🆕 有更新：
1. 猫笔刀 — 最后更新: 2025-05-30  [打开](weread://reading?bId=MP_WXS_3198966508)

✅ 无新内容：
2. 重燃阅读 — 最后更新: 2025-05-29  [打开](weread://reading?bId=MP_WXS_2392369754)
```

3. 如果有 `new` 状态的公众号，用 🆕 标记并提示"新关注的公众号"
4. 如果有 `removed` 状态的公众号，提示"以下公众号已不在书架上：xxx"

### 2. 查看详情

**触发词**：用户点击/选择某个公众号，或说"看看猫笔刀的详情"

**工作流**：

1. 执行 `python3 scripts/detail.py --name "公众号名称"` 或 `python3 scripts/detail.py --id MP_WXS_xxxx`
2. 按以下格式展示：

```
📰 猫笔刀

简介：记录与分享！
最后更新：2025-05-30
最后阅读：2025-06-01
状态：有未读内容

[打开阅读](weread://reading?bId=MP_WXS_3198966508)
```

## 展示规范

- **时间戳**：所有 Unix 时间戳展示为 YYYY-MM-DD 格式
- **深度链接**：每个公众号附上 `weread://reading?bId={bookId}` 跳转链接
- **状态标记**：🆕 有更新 / ✅ 无新内容 / 📌 新关注 / ❌ 已取关
